# Дневник обучения — 5 июля 2026

## S1 — Резидентура, сессия 1 (2026-07-05, 11:00 MSK)

**Заметка — "Что такое важное?"**

Три критерия истинной важности — если хотя бы одного нет, значит "важное" декларативное, а не реальное:

1. **Важное должно быть в плане** — если не вошло в WeekPlan/DayPlan, оно не важное, оно просто желаемое
2. **Важное должно иметь рабочие продукты** — конкретный измеримый результат, не процесс ("работал над X") а артефакт ("создал Y")
3. **Важное должно быть забюджетировано** — выделено реальное время (часы), не "найду время когда появится"

**Вывод:** "Важное" без плана + РП + бюджета = самообман. Это не приоритет, это wishlist.

**Связь с моим контекстом:**
- Ataccama Data Trust — есть Roadmap v0.1, но нет бюджета времени в WeekPlan на эту неделю (сознательно отложено) — значит пока честно не важное
- Книжный клуб (идея из inbox) — нет ни плана, ни РП, ни бюджета → wishlist, пока не важное
- S1/S2 — есть план (DayPlan), есть РП (читаю главы), есть бюджет (2×30мин) → реально важное
