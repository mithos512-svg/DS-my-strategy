# Day Rhythm Config — протоколы и правила IWE

## Стандарты обозначений

**В DayPlan/WeekPlan — приоритеты РП:**
- P1 — критический, выполнить обязательно
- P2 — средний, важно но допустим перенос
- P3 — поддерживающий (спорт, здоровье, ритм)

**В nutrition-логе — достижение нормы:**
- 🟢 в норме / 🟡 пограничная зона / 🔴 недобор или превышение

- **Спортзал** — вт, пт, вс (всегда утром, первым РП дня)

---

## Day Open — чек-лист

1. Записать вес → обновить `current/weight-log.md` ЦЕЛИКОМ
2. Проверить день недели → добавить спорт по расписанию
3. Проверить inbox
4. Собрать DayPlan → `current/DayPlan {дата}.md`
5. Поставить Telegram на Day Close (отправлять на **19:30**)

---

## Day Close — чек-лист

1. Итоги дня — что выполнено/перенесено
2. Финальный nutrition-итог дня
3. Append learning-journal дня в общий архив
4. DayPlan → `archive/day-plans/`
5. Forced reindex: `IWE:personal_reindex`
6. Поставить Telegram на Day Open завтра (отправлять на **8:00**)
7. Если завтра воскресенье → "Week Close + Strategy Session"

---

## Strategy Session (воскресенье) — чек-лист

1. Week Close — итоги недели по РП
2. Nutrition — сводка за неделю (ккал/белок/клетчатка/вес)
3. Разбор заметок → `current/posts/ideas.md` → выбрать 1 тему
4. Разбор inbox → РП
5. WeekPlan следующей недели
6. IWE Roadmap — прогресс по 14 элементам

**Месячная Session** — первое воскресенье месяца (~2ч). Ближайшая: 2 августа 2026.

## Nutrition — протокол

**Нормы:** калории ≤2000 (алерт 1800), белок 160г, клетчатка 25-30г

**Шаблоны:**
- "завтрак стандартный" → творог 5% + банан + кофе = ~334 ккал, ~24.8г белка
- "каша стандартная" → овсянка + голубика 35г + груша ½ + кофе = ~305 ккал, ~9.1г белка
- "полдник стандартный" → Protein Pudding + черника 96г + морковь + груша + Bar = ~488 ккал, ~37.8г белка

**Week Close:** средние ккал/белок/клетчатка + тренд веса за неделю.

## Правила хранения файлов

Файлы, которые растут (weight-log, fleeting-notes, learning-journal архив) → перезаписывать ЦЕЛИКОМ при каждом обновлении. Это гарантирует что `personal_get_document` вернёт полный файл, а не последний чанк.
