# Day Rhythm Config - протоколы и правила IWE

## Стандарты обозначений

**В DayPlan/WeekPlan - приоритеты РП:**
- P1 - критический, выполнить обязательно
- P2 - средний, важно но допустим перенос
- P3 - поддерживающий (спорт, здоровье, ритм)

**В nutrition-логе - достижение нормы:**
- В норме / пограничная зона / недобор или превышение

**СВД** = стандартный вечерний десерт (сокращение, добавлено 20.07)

---

## Суббота - работа над постом в клуб МИМ (добавлено 18.07)

**Регулярный РП по субботам:** работа над черновиком поста для клуба МИМ
(разбор заметок недели -> черновик -> редактура). Приоритет P2.

Добавлять в DayPlan субботы по умолчанию, если нет explicit других приоритетов.

---

## Контрольный вопрос FPF - "что открывает следующий ход" (добавлено 16.07)

При формулировке ключевых РП в WeekPlan (особенно P1) - задавать явный вопрос:
"Что можно будет сделать после завершения этого РП?"

## Замеры тела - US Navy (еженедельно)

**Формула % жира (мужчины):**
% жира = 495 / (1.0324 - 0.19077 x log10(талия - шея) + 0.15456 x log10(рост)) - 450

**Протокол:**
- При Day Close в субботу - напомнить про замеры талии и шеи
- При Day Open в воскресенье - спросить обхват талии и шеи, посчитать % жира US Navy
- Записать в current/weight-log.md рядом с весом

**Данные:** рост 182 см

## Day Open - чек-лист

1. Записать вес - обновить current/weight-log.md ЦЕЛИКОМ
2. Если воскресенье - спросить замеры талии и шеи - посчитать % жира US Navy
3. Проверить день недели - добавить спорт по расписанию
4. Если суббота - добавить РП "работа над постом в клуб МИМ" (P2)
5. Показать текущий WeekPlan (РП недели со статусами)
6. Проверить inbox
7. Собрать DayPlan - current/DayPlan {дата}.md
8. Поставить Telegram на Day Close (отправлять на 19:30)

---

## Day Close - чек-лист

1. Итоги дня - что выполнено/перенесено
2. Показать текущий WeekPlan - обновить статусы
3. Финальный nutrition-итог дня
4. Если суббота - напомнить про замеры талии и шеи
5. Append learning-journal дня в общий архив
6. DayPlan - archive/day-plans/
7. Forced reindex
8. Поставить Telegram на Day Open завтра (отправлять на 8:00)
9. Если завтра воскресенье - "Week Close + Strategy Session + замеры тела"

---

## Strategy Session (воскресенье) - чек-лист

1. Week Close - итоги недели по РП
2. Nutrition - сводка за неделю
3. Разбор заметок - current/posts/ideas.md - выбрать 1 тему
4. Разбор inbox - РП
5. WeekPlan следующей недели - для ключевых РП задавать вопрос "что это открывает"
6. IWE Roadmap - прогресс по 14 элементам

Месячная Session - первое воскресенье месяца. Ближайшая: 2 августа 2026.

## Nutrition - протокол

Нормы: калории 2000 макс (алерт 1800), белок 160г, клетчатка 25-30г

Шаблоны:
- "завтрак стандартный" - творог 5% + банан + кофе = ~334 ккал, ~24.8г белка
- "каша стандартная" - овсянка + голубика 35г + груша половина + кофе = ~305 ккал, ~9.1г белка
- "полдник стандартный" - Protein Pudding + черника 96г + морковь + груша + Bar = ~488 ккал, ~37.8г белка
- "СВД" (стандартный вечерний десерт) - шоколад + орехи + чернослив = ~286 ккал, ~4.7г белка

Week Close: средние ккал/белок/клетчатка + тренд веса + % жира за неделю.

## Практика самопроверки через проговаривание (добавлено 15.07)

Формат: пользователь диктует принцип и свой ответ - Claude даёт обратную связь -
уточнения фиксируются - финальная проверка связного объяснения своими словами
плюс пример. Критерий "могу объяснить" = свои слова + пример, не заучивание наизусть.

Расширение (20.07): формат также применим к проверке понятий (не только таблиц
принципов) - роль/должность, границы систем и т.д. И применим к разбору личных
вопросов/неудовлетворённостей (роли в конфликтах, антихрупкость - оба разобраны
через этот формат).
